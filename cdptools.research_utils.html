cdptools.research_utils.transcripts.download_most_recent_transcripts(db: cdptools.databases.database.Database, fs: cdptools.file_stores.file_store.FileStore, save_dir: Union[str, pathlib.Path, None] = None) → Dict[str, pathlib.Path][source]¶

Download the most recent versions of event transcripts.

Parameters

  • db (Database) – An already initialized database object connected to a CDP instance’s database.

  • fs (FileStore) – An already initialized file store object connected to a CDP instance’s file store.

  • save_dir (Optional[Union[str, Path]]) – An optional path of where to save the transcripts and manifest CSV. If None provided, uses current directory. Always overwrites existing transcripts with the same name if they already exist in the provided directory.

Returns

event_corpus_map – A dictionary mapping event id to local Path of the most recent transcript for that event.

Return type

Dict[str, Path]

Get a pandas dataframe that can act as a manifest of the most recent transcript available for each event stored in a CDP instance’s database.

Parameters

db (Database) – An already initialized database object connected to a CDP instance’s database.

Returns

manifest – A dataframe with transcript, event, body, and file details where each row is the most recent transcript for the event of that row.

pandas.DataFrame
